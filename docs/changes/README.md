# 渊默雷声博客变更记录

追加式、有序、不可删改的操作记录。

## 目录结构

```
docs/changes/
  arch/   架构决策类  ← 为什么做、怎么设计的
```

## 命名规则

```
NNN_kebab-case-description.md
```

- 每个目录独立编号，从 `001` 开始
- 追加式：只增不删
